--About Concentration--

1. It is a web game.


--Getting started--

1. clone the folder

2. Open concentration.html in browser

3. You will see block containing 16 cards.

4. click to open cards.

5. If matched will turn green.

6. If not matched will turn to red and hide again.

7. Rest button to shuffle the cards and restart again.

8. Timer will start as soon as you refresh browser or click reset button.

9. Rating will be given based on the number of moves.

10. Once you complete the game your timer will stop and you will get a 'congratulation' message.
